## Platform Features

The TonightPass platform features real-time ticketing, seamless event management, and powerful organization tools, designed to bring people together through unforgettable experiences.

- **For organizers**: Everything needed to create, manage, and grow your events.
- **For attendees**: A modern way to discover and access the best events in your city.

## SDK Packages

TonightPass provides several SDK packages to help you integrate with the platform:

- **tonightpass**: Core Node.js SDK for interacting with the TonightPass API
- **@tonightpass/react**: React hooks and components for building client-side applications
- **@tonightpass/nest**: NestJS integration for server-side applications

For detailed API documentation, check out the [API Reference](/api-reference).
